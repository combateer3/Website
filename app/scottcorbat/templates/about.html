This website is developed by Scott Corbat, a student at Michigan Technological University.
It makes use of several docker containers running on a homemade server PC. Webpages are designed using html, css, and flask-bootstrap. A Traefik container provides a reverse proxy service and automated SSL certificates.

Service Version/Tag Description
Flask-UWSGI-Nginx python3.6 My docker container is built on this one to use flask to serve webpages behind UWSGI for heavier server loads and Nginx as a reverse proxy.
Nextcloud latest A self-hosted storage solution that can be accessed from anywhere, including a mobile application.
Traefik latest A reverse proxy for docker containers. Also provides automated SSL certification.
MariaDB latest The database connected to the Nextcloud container.
Minecraft Server 1.12.2 Modded A self-hosted server for the popular game Minecraft Java Edition. Address is 'mc.scottcorbat.com'
