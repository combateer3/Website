The Services

This website makes use of several docker containers running on a homemade server PC. Webpages are designed using html, css, and flask-bootstrap. A Traefik container provides a reverse proxy service and automated SSL certificates.

Service Version/Tag Description
Flask-UWSGI-Nginx python3.6 My docker container is built on this one to use flask to serve webpages behind UWSGI for heavier server loads and Nginx as a reverse proxy. Source code for my Flask application can be found on my GitHub.
Nextcloud latest A self-hosted storage solution that can be accessed from anywhere, including a mobile application.
Traefik latest A reverse proxy for docker containers. Also provides automated SSL certification.
MariaDB latest The database connected to the Nextcloud container.
Minecraft Server 1.14.4 A self-hosted server for the popular game Minecraft Java Edition. More info here.

The Server

All services are hosted on a low budget computer built by myself for a few reasons. I enjoy building computers as a pasttime. This also removes dependencies on third parties, with the exception of my ISP. Finally, it gives me experience setting up my own web server from scratch.
